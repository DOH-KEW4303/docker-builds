# ngmaster 1.0.0 container

Main tool : [ngmaster](https://github.com/MDU-PHL/ngmaster)

In silico multi-antigen sequence typing for Neisseria gonorrhoeae (NG-MAST).

Additional tools:

- any2fasta 0.4.2
- mlst 2.23.0
- python 3.7.12
- biopython 1.79
- perl 5.32.1
- bioperl 1.7.8

## Example Usage

```bash
$ ngmaster gono.fasta
ID         NG-MAST    POR    TBPB
gono.fa    10699      6277   4
```
